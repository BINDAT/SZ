At Start-Zup, a free school, we had a core module that covered the history of IT in all its forms and the right reflexes to adopt, and then we designed a Wordpress site for real customers.

In the second module we looked at the following code languages:

  • Html
  • CSS
  • JAVASCRIPT
  • PHP
  • SQL

All this with some useful tools like Visual Studio Github and many others you can discover. Personally this my favorite module

Then there was the third module on ITSM Service Now and the Microsoft 365 solution, which I didn't enjoy as much, but which was still a lot of fun. In this last module, you'll learn how to use Service Now to create service incident or change tickets, track their progress as the requestor creates them, and manage them. For Microsoft 365, which I liked a little more, you'll learn how to

  • create an infranet site with Share Point for example,
  • or manage team groups,
  • create mailing lists or create users.

But above all, the school will give you the opportunity to take up internships with companies of all types and with partners of all types, or even to continue with the school as a trainee, so that you can continue your training. The school also aims to create its own applications and to grow and take on new, motivated people, even if you're not good at programming or anything else, as long as you're captivated by IT and, above all, are motivated and present, whether face-to-face or distance learning, it's all ok.
